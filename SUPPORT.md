# Support

KnowMesh is an early public-alpha open-source project. The fastest way to help is to report reproducible behavior with enough local context for maintainers to verify it without seeing your private documents.

## Before Opening an Issue

- Check `README.md`, `README.en.md`, `docs/README.md`, and `docs/current-design.md`.
- Run `npm test`, `npm run smoke:release`, `npm run smoke:artifact`, `npm run verify:package-boundary`, or `npm run doctor` when relevant.
- Remove secrets, local document text, generated artifacts, and private paths from logs.

## Where To Ask

- Reproducible bugs: open a bug report issue.
- Product or workflow requests: open a feature request issue.
- Documentation problems: open a documentation issue.
- Roadmap and design discussions: use GitHub Discussions when enabled.
- Security vulnerabilities: report privately as described in `SECURITY.md`.

## Not Included

The public repository must not contain user workspaces, source document corpora, generated SQLite databases, cloud credentials, or private processing artifacts.
