# Support

KnowMesh is currently in early private development on GitHub, with the intent to become public open source.

## Before Opening an Issue

- Check `README.md`, `docs/current-design.md`, and `docs/phase1-6-operations-runbook.md`.
- Run `npm test`, `npm run smoke:release`, or `npm run doctor` when relevant.
- Remove secrets, local document text, generated artifacts, and private paths from logs.

## Where To Ask

- Reproducible bugs: open a bug report issue.
- Product or workflow requests: open a feature request issue.
- Security vulnerabilities: report privately as described in `SECURITY.md`.

## Not Included

The public repository should not contain user workspaces, source document corpora, generated SQLite databases, cloud credentials, or private processing artifacts.
