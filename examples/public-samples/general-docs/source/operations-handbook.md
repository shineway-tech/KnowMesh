# Public Sample Operations Handbook

This is a fictional public sample for KnowMesh. It is not a private customer document.

## Knowledge Asset Rules

Every published knowledge asset must have a source owner, a weekly review cadence, rollback-ready releases, and cited answers. The source owner confirms whether a document should stay active, enter review, or be archived.

## Query Runtime Acceptance

The public sample is considered queryable only when an answer can cite the source document and page. If a question asks about a topic that is not present in the sample, KnowMesh should return a no-answer state instead of inventing details.

## Maintenance Loop

Feedback should become reviewable maintenance work. Release manifests must stay redacted: they can expose checksums, counts, and paths, but not private source text or credentials.
